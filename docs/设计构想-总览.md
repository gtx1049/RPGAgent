# RPGAgent 设计总览

> 归档时间：2026-03-24
> 版本：v0.1 构思阶段

---

## 项目愿景

用大模型上下文能力做 RPG 游戏引擎，AI 充当 DM（地下城主），玩家用自然语言进行游戏。

核心差异化方向：
- **隐藏数值系统**：不是显示血条，是让玩家在叙事中感受到角色的变化
- **长记忆 + 跨场景因果**：LLM 记住所有行为和沉默，债务跨章节清算
- **NPC 作为真实存在**：每个 NPC 有独立记忆池，人物关系可被破坏/传播

---

## 参考灵感

- D&D 跑团（核心体验）
- AgentScope（底层 Agent 框架）
- 传统 RPG 数值系统（可验证性）

---

## 设计原则

1. **规则可验证**：数值逻辑走 Python 代码，LLM 只负责叙事
2. **沉浸优先**：隐藏数值不显示具体数字，而是改变叙事体验
3. **记忆即资产**：对话历史、行为记录、沉默记录都是游戏资产
4. **剧本即配置**：同一引擎，配不同剧本变成不同游戏
