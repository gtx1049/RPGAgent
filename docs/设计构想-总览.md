> 归档时间：2026-03-24
> 更新时间：2026-03-26

---

## 项目愿景

用大模型上下文能力做 RPG 游戏引擎，AI 充当 DM（地下城主），玩家用自然语言进行游戏。

核心差异化方向：
- **隐藏数值系统**：不是显示血条，是让玩家在叙事中感受到角色的变化
- **长记忆 + 跨场景因果**：LLM 记住所有行为和沉默，债务跨章节清算
- **NPC 作为真实存在**：每个 NPC 有独立记忆池，人物关系可被破坏/传播

---

## 参考灵感

- D&D 跑团（核心体验）
- AgentScope（底层 Agent 框架）
- 传统 RPG 数值系统（可验证性）
- 博德之门 / 永恒之柱（数值成长参考）

---

## 设计原则

1. **规则可验证**：数值逻辑走 Python 代码，LLM 只负责叙事
2. **沉浸优先**：隐藏数值不显示具体数字，而是改变叙事体验
3. **记忆即资产**：对话历史、行为记录、沉默记录都是游戏资产
4. **剧本即配置**：同一引擎，配不同剧本变成不同游戏

## 数值系统详解

### 属性修正（d20 规则）

```
属性修正 = (属性值 - 10) / 2（向下取整）

  属性值   修正
  1-2     -5
  3-4     -3
  5-6     -2
  7-8     -1
  9-10     0
  11-12   +1
  13-14   +2
  15-16   +3
  17-18   +4
  ...
```

### 六属性

| 属性 | 英文 | 影响 |
|------|------|------|
| 力量 | Strength | 近战伤害、攀爬、力量检定 |
| 敏捷 | Dexterity | 闪避、潜行、远程命中 |
| 体质 | Constitution | HP上限、毒素抵抗 |
| 智力 | Intelligence | 学识技能、魔法效果 |
| 感知 | Wisdom | 侦查、感知、意志检定 |
| 魅力 | Charisma | 社交、威吓、欺骗 |

### 难度档位（显示给玩家）

| 档位 | 成功率 | 示例 |
|------|---------|------|
| 轻松 | ≥70% | 日常对话、轻松任务 |
| 五五开 | 40-69% | 有风险行动 |
| 难搞 | 20-39% | 困难任务 |
| 几无可能 | <20% | 近乎不可能 |

> 内部仍基于 d100 + 属性修正计算，玩家只看到这 4 个直观档位。

### 行动力

- 每回合开始恢复至最大（默认 3 点）
- 普通行动消耗 1 点行动力
- 特殊行动（强力攻击等）消耗更多
- 行动力耗尽时只能执行免费动作（观察、说话）

### 技能点

- 每次升级获得 2 点技能点
- 技能最高 5 级（部分 3 级）
- 被动技能常驻生效
- 主动技能消耗 AP 释放

### 装备稀有度

| 稀有度 | 颜色 | 效果强度 |
|--------|------|----------|
| Common | 白色 | 基础 |
| Uncommon | 绿色 | +1~2 属性 |
| Rare | 蓝色 | +3 属性 / 技能 |
| Epic | 紫色 | +4~5 属性 / 特殊效果 |
| Legendary | 橙色 | 传奇效果 |
